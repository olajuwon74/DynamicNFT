9. DATA PROCESSING STRATEGY

When app users use Amemori to perform image restoration, there may be a certain waiting time in obtaining carousel information and image processing. In order to reduce the data response time and improve user experience, the system performs data processing in multiple aspects.

Using caching technology, only when the user uses the app for the first time, the data will be obtained in real time, and then it will be obtained from the cache to improve the response speed. Only when the data is updated, will the data be obtained in real time again.

During the image processing, it may take some time to wait. The system uses the RabbitMQ message queue to notify the app user of the processing progress. The user does not need to stay in the processing interface and wait, and can continue to perform other image processing or do other things. After the image processing is completed, the system will automatically notify the user that the processing is complete.
